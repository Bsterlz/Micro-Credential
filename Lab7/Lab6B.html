Lab: 6 Favicons!


What exactly is a favicon?

Also known as a shortcut icon, website icon, tab icon, URL icon, or bookmark icon, a favicon is a file containing one or more small icons, associated with a particular website or web page.

How are favicons created?

First you wanna create your image using binary code with the first 16 bytes controlling your width and length. The next bytes will define your color and you must also define bytes per pixel to create different shades of colors. Then save and convert your image and then upload it to your website.

How did I create the favicon for this site?

I first created a 14x14 pixel with my first 16 bytes. I had in mind that I wanted the logo to represent my first intial. So I decided to use 6 bytes per pixel and every square until a B was shaped.
